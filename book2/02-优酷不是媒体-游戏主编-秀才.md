# 二 优酷不是媒体 游戏主编 秀才

> 首发于知乎专栏（2014-09-13）原文链接：https://zhuanlan.zhihu.com/p/19847893

中国游戏幕后史二 优酷不是媒体 游戏主编 秀才
文/BBKinG

　　秀才（周晔），现优酷游戏主编 ，1981年7月14日出生于江苏无锡，从小喜欢文学和历史，家中的藏书很多，涉猎广泛，尤其擅长文言文阅读，酷爱《三国演义》，“秀才”这个外号也是因此而来。

　　突发奇想，考入上戏导演系

　　2000年，高考前夕，秀才看了一个电视剧，故事背景是发生在上海戏剧学院，他看的很入迷，突然觉得这个学校很有意思，从而萌发了报考这个学校的念头。

　　可是戏剧学院里都是艺术专业，而他当时是理科生，“理科生是否能报考？” 成了他说服家长后面对的第一个问题，后来经过询问，发现原来高考志愿表上有一个选项是 “是否艺兼理”，只要把这个选上就可以报考了，于是，在经过一番折腾后，秀才成了无锡那所中学建校百年来第一个转艺术的理科生。

　　艺术考试是很难的，“可以考”只是找到了门而已。

　　考什么专业呢？秀才抱着上海戏剧学院的招生简章看来看去。

　　艺术专业大多数都是需要多年专业培训的，翻来覆去的比较，他最后发觉只有一个专业似乎适合自己——导演系。因为导演系的报考条件看起来最为泛泛，似乎什么都需要懂一点，但是不需要精通。

　　秀才自嘲的说，什么都不会的就去考导演系了。

　　可是，如果你真以为导演系好考就错大了，标准越模糊的东西越难琢磨。那一年全国有近2000人报上戏的导演系，最后只录取了16个人，而且考导演系的大多都是有丰富社会阅历和影视工作经验的人，这些人年龄也比较大，秀才这种应届高中生跟一群中年人拼，胜算实在很低。

　　而且，考导演系是要过三试的。

　　一试是考具象命题小品，就是给你十道比较具体的命题，你选一个，准备15分钟，演一段5分钟的小品，这个对于艺术专业的学生会比较简单，但是对于秀才来说，这个部分是最难的，他没受过这方面训练，不过所幸，他勉强过了。

　　二试是考抽象命题小品，秀才拿到的题目是“黑色”，看到这里你可以先停一下，想想如果是你拿到了这个题目，你会怎么演？

　　二试还有抽十道各种天马行空的问题让你回答，看你的知识面和社会阅历如何。本来这对于秀才来说是一个很大的考验，毕竟年龄不大，不过，读书广泛这个爱好帮了他很大的忙。

　　下面说说秀才拿到“黑色”这个题目后是怎么演的。

　　他很快的准备了一下，然后演了一段 “高三晚自习时突然停电” 的小品，把停电前全班埋头疾书，忙碌而压抑，停电后的茫然、黑暗中的尖叫和释放，来电后的尴尬，恢复忙碌压抑，用一个不到5分钟的小品展现了出来，快速的捕捉到了一段细微却很有张力的生活瞬间。

　　三试是考写影评，现场放了部电影，冯巩主演的《埋伏》，看完后马上写，这个对他也不是太难。

　　最终，秀才以全国第五名的成绩考进了上海戏剧学院导演系。进入这所上世纪40年代就成立的艺术大学。

　　很多人看到这里会问：你写这些这跟游戏幕后史有关吗？

　　有没有关系，暂且不表，你接着往下看。

　　开学了，秀才进入了上戏大一，16个同学被分成2个班，每个老师带8个人上课，学习的是所有跟影视有关的专业，只是没有各个专业那么深，大一大二学表演，大三大四跟其它专业的人一起上课，要求就是什么都懂，但不深钻，导演专业认为，掌控大局的人要做通才而不是专才，训练总经理和军官都是如此。

　　于是，导演系毕业的人出来干什么影视工种都可以，秀才说，现在很多实力派演员都是导演系毕业的，但是反过来就很困难了，人有时候会被自己擅长的东西所限制。

　　上海戏剧学院每年大四毕业的人会一起出一个毕业大戏，导演系和表演系各出一个，两边分别带着同一届的其它专业合作完成，这是每年的一个重头戏。

　　等大四忙完毕业大戏，秀才一个班16个人只剩下12个了，淘汰率是很残酷的。

　　与游戏的结缘

　　秀才在大一这年因为爱好游戏，认识大他一届的张锐（现游戏风云节目制片人），这为他后来进入游戏风云埋下了伏笔。

　　2002年，秀才被灯光专业的同学拖着玩《RO》，之后是《魔剑》《UO》，认识了很多游戏圈的朋友。

　　关于游戏圈，秀才认为，2003年，在《UO》这款游戏里诞生了中国第一批公会玩家，在《魔剑》 诞生了中国第二批公会玩家，在《天堂2》和《WOW》里走在一起的算第三批，有很多现在的游戏行业领导者都是在前两个时期就比较出名的人，时势造英雄。

　　Channel[G]游戏频道　　

　　2004年毕业后，秀才进入了明日科技公司下的Channel[G]游戏频道负责网游资讯和电竞节目，当时这个频道是给上海文广互动电视的游戏风云频道供片的。

　　在这个时期，秀才认识了妖魔（前游戏风云副总经理），孙俊（游戏风云网站部主管），吴强和文圣可等等后来GamesTV的几个创始人。

　　加盟GamesTV

　　2005年Chinajoy之后，秀才在张锐的引荐下加入了GamesTV（后来与游戏风云合并），负责网游节目，张锐负责单机游戏节目，妖魔负责电竞节目。

　　在2007年之前，秀才一直是做编导，做了WOW的攻略、专题、文化渊源的节目，2007年之后转去做销售，因为毕竟导演出身，什么都懂，也熟悉游戏，所以在销售方面做的还不错。

　　这个时期直播技术已经有了比较大的发展，PPS和PPLIVE之类的直播平台已经出现，MediaCode流媒体直播技术开始成熟，内容生产商只用把直播流推到PPS等专业直播平台，他们就可以满足大规模用户同时在线观看的需求。

　　2006年初，视频直播平台成功向全中国直播的StarsWar2，20万独立IP，200多万用户收看，就是在这样一个技术突破的背景下获得巨大成功的案例。

　　秀才认为，技术革命带动了电子竞技的几次高速增长

　　从1998年到2005年之前如此漫长的一段时间里，中国是无法同步看到大型电子竞技比赛的直播画面的，依靠的都是图文报道，视频直播在全世界都是一个技术瓶颈，甚至有一段时间被逼出了几个音频直播网站。

　　秀才说，2004年大家做直播是用微软的Mediaplay，这是早期视频直播比较无奈的做法，因为如果是1M的码流往外传输，有1个人看，那带宽就需要1M，2个人看就是2M，如果是100人就是100M，以当时的带宽普及程度和高资费标准，谁都无法支持这种大规模的视频直播，而且最重要的是这个技术也支持不了多少人同时在线观看。

　　大众对内容的需求促使了技术的变革，技术的变革扩大了内容的影响力，游戏历史几乎就是在被这2个部分交替着往前推动着。

　　优酷游戏的技术突破

　　2006年优酷成立了，由于他们在不断的改善CDN网络加速技术在优酷中的应用，这使得优酷流媒体点播流畅度在不断提高，也正是在这个大背景下，中国游戏格局，高速宽带的普及度，电竞格局都发生了巨大的变化。

　　2008年，优酷开始在上海建立分公司时（之前在北京），就接触了到了秀才。

　　刚好秀才此时对于行业的理解也发生了很大的转变。

　　在2008年之前，他觉得在视频行业里是内容为王，就是生产节目的人占主导，但是在2008年之后，他发觉是平台为王，就是播出渠道占主导。于是他开始寻找合适的平台做合作。

　　他发现，土豆网当时还没有进行专门的梳理，他们的游戏频道是在用户上传后自动生成的，而优酷已经开始做内容筛选了，不过，土豆网当时做的比优酷网早，也比优酷大的多，用户已经习惯了土豆网，用户习惯，有时是一座很难改变的大山。

　　加盟优酷

　　2009年，秀才在加入优酷后，负责优酷游戏行业的营销策划。2009年6月，在原优酷游戏主编被土豆挖走后，秀才成为游戏频道主编。

　　秀才说，刚上任时，摆在他面前的是一片废墟，面对当时的土豆，优酷毫无优势，竞争对手不但内容更多，而且达人聚集，优酷游戏才刚开始，再加上原优酷的人又走了一批游戏资源过去，秀才只好另辟蹊径。

　　优酷游戏从电竞节目再起步

　　因为秀才之前在游戏风云时跟电子竞技方面经常接触，所以在经过一番分析和论证后，他决定避开网游，从电竞节目方面开始入手，特别是2009年DOTA正处于上升期，用户的教学视频等需求在不断上升。

　　于是，秀才将现有的DOTA节目作者进行梳理，觉得有2个人有一定的用户基础（水离子和加菲盐），后来又挖掘一些新的作者，比如DZ，海涛，把游戏频道的推荐资源都用在他们的身上，再利用一些电竞方面老关系，比如[http://Replays.net](http://link.zhihu.com/?target=http%3A//Replays.net)等网站进行联合推广，这个雪球开始慢慢越滚越大了。

　　半年后，秀才做的优酷金牌解说团队从最开始的5个人扩大到了10个人，这个时候正是DOTA在中国逐渐兴起的时候，不知道是游戏推动了视频，还是视频推动了游戏，2010年解说视频的点播量也开始有了显著的提高，从最开始的几万到了十几万，用户慢慢养成了看比赛视频的习惯。

　　以至于，海涛在2010年年底开始在优酷做视频时，仅仅用了三四个月就到了三十万，半年后的2009更是直接就站在了50万点击的起步门槛上，这些都和之前的逐步铺垫有着密切的关系。

　　天时地利人和，命运的推波助澜

　　2010年12月8日，优酷上市纽约证券交易所，充足的资金让优酷的发展更是如虎添翼，视频流畅度和用户体验度不断提高，在DOTA继续火热的同时，秀才已经暗地开始了《英雄联盟》的储备。

　　优酷游戏吸引了大量的新晋解说，他们也通过优酷这个平台获得了大量的关注度，随着淘宝等解说商业化行为的完善，解说收入的稳步和制作门槛的降低，吸引了更多人加入到这个行当里来，于是就有了从2010年到现在的电子竞技高速发展。

　　在大时代的背景下，很多人在对的时间、对的地点、做了对的事，就起来了，包括秀才自己。

　　五年以来，从优酷成长起来的知名解说不计其数，其中的佼佼者例如海涛、2009、小智、小苍、小漠等在起步的时候都得到过秀才的诸多帮助。

　　优酷是一个平台而不是媒体

　　可以说，电子竞技在视频上的巨大需求以及优酷在视频资源上的储备，二者结合极大地推动了电竞的发展，而解说这一全新的职业也藉由着这个平台全新崛起，形成了观众与电竞选手和赛事间的纽带。随着这个环节的打通，以往电子竞技关注度变现问题也逐步得到解决，而解说也成为更多的职业选手退役之后最可靠的去处。

　　然而，秀才也认为，优酷并不是一个媒体，而是一个让每位解说得到展现的舞台，但是解说这种自媒体的崛起，会在很大程度上削弱媒体的话语权，甚至让媒体依赖于解说创造的内容，大解说的影响力超越一些电竞网站也是不争的事实，这一点众多的电子竞技垂直类网站更深有感触。

　　如何平衡媒体、解说、平台之间的关系？是这种业态成长起来之后的新课题。

　　新的技术瓶颈

　　秀才说，面对自媒体的大趋势，优酷游戏一方面在做着积极的调整，并看好移动终端的爆发点，但是依然有很多问题摆在面前，比如，如何解决移动网络提速，流量费用昂贵，观众对内容需求更加休闲化，对新闻报道速度要求更高等等问题上，而这些问题又卡在了新的技术瓶颈上。

　　换句话说，新的技术变革可能也要呼之欲出了。

　　谁能最先解决这个技术瓶颈，占尽这个先机呢？

　　会是谁呢？
