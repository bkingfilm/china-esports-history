# 二十二 WE.LOL队长 若风Misaya

> 首发于知乎专栏（2013-12-12）原文链接：https://zhuanlan.zhihu.com/p/19635952

中国电子竞技幕后史二十二 WE.LOL队长 若风Misaya  文/BBKinG

　　2007年的一天，若风又一次跟他父母因为学业的事情吵架，这一次吵的比以前更加激烈，以至于他猛的独自冲出家门，跑出很远，走在一条很长的路上，那天风很大，他一边走，一边想如果自己若能像风一样该多好，自由自在，无拘无束。

　　时光飞逝到5年后，2012年10月28日，中国最著名的电子竞技俱乐部英雄联盟战队队长若风和队友们一起站在WEM的冠军领奖台上，这一次他没有哭，甚至有些平静，赛后很谦逊的说通过比赛发现很多不足，会跟队友一起改进，去争取更多的冠军和荣誉。

　　一个经过无数次失败和爬起的斗士终于开始成熟了，但是在这之前他和其它电子竞技选手一样经历了很多家人的不理解和艰辛的磨练。

　　叛逆

　　若风，1992年10月19日出生于湖南邵东，后因父母工作全家迁到湘潭，若风的英文名叫Misaya，中文名，禹景曦，但是直到他升入高中，他都是用另一个中文名，禹竞雄。后来他母亲觉得他叛逆心太重可能跟名字有关，才改成了现在的禹景曦。

　　高一的一次学校事件让若风产生了很深的厌学感，甚至发生了几次离家出走。这让他的母亲十分焦急，无奈之下只好帮若风转到另一所学校，但是到新学校仅仅2个月后的一天，因为若风忘记穿校服而被年级主任在校门口大骂，并且特别针对他的转校生身份，从头骂到尾，没有给若风任何解释的机会，这一次彻底撕裂了若风的尊严，他再也没有回到那所学校。

　　2008年的时候，若风很迷茫，他喜欢足球，游戏，文学，但是社会不认同这些，考试也不认同这些，于是他对学业方面也彻底失去了兴趣，没有去上学了。

　　父母的失望和自己的倔强

　　父亲是大学教授博士后，所以从小就对他有着很高的期望，但是那段时间里，父亲对若风失望到了极点。父子之间的关系几乎破裂。

　　母亲是中学英语老师，为了他真是操碎了心，一边希望他能回头，一边又不敢太逼他，因为怕他会离家出走，所以买了台电脑不让若风去网吧，那段时间他非常压抑，为了证明自己选的这条路是对的，若风整天在家里疯狂的练习DOTA，吃饭都尽量避开父母的目光，每天想的就是自己怎样能变强，就这样一打就是2年。

　　若风带着变强的愿望开始了疯狂的训练，每天只有2到3个小时的睡眠，不断的看DOTA比赛录像和教程，还在网上认识了湖南天马战队的队长Aayaya，并通过自己在训练赛中的表现，成为了一名网上的队员，有段时间甚至包揽了湖南本地的所有奖项。

　　两次职业之路的失败

　　08年的时候，武汉的一个人说要组建职业战队，当时DOTA的职业战队很少，若风又很想打职业，就带了自己攒的1000块压岁钱就过去了，后来那人说要交房租什么的，没带现金，若风当时只有16岁，觉得既然来了就该信任人家，就把钱给那人了，后来那人就消失了。若风最后很尴尬的到处找人借钱才回到家。

　　回家后，让若风很意外的是，母亲并没有责备他，而是觉得他是很认真的在做这件事情，只是事情的结果不太好而已，于是从那时开始鼓励他。

　　第一次职业之路的失败并没有让若风停止自己的梦想，在2009年8月，他认识了著名DOTA选手ZSMJ，并成为很好的朋友，10月的一天ZSMJ告诉他一个好消息，希望他跟自己一起加入后来有九神之称的伍声2009组建的FTD战队，在经过考核后，ZSMJ和若风成为FTD战队最后的2名队员，来到杭州开始了职业训练。

　　当时FTD可能是为了整体实力强一些，于是找了5个打主力的选手组成的，但是开始训练的时候就出现了麻烦，5个人没人愿意打辅助位置，最后若风被安排打辅助，当时他心里是不太好受的，因为他没打过这个位置，也打的不太好，但是团队有分工，他就去练恶魔巫师，为了练好辅助，那段时间请教了很多人，每天只练这一个英雄，但是一个月后，依然不是太好，当时很沮丧，觉得自己拖战队后腿了，就给队友们留了封信，离开了。

　　2次的职业之路受阻，对若风的打击很大，但是他依然没有放弃训练，终于在2010年WCG，若风等来了一个机会，TOT战队的chuan因为是外国国籍，不能参加中国区WCG，于是把若风火线招入，代替chuan打主力的位置，在苏州集中训练了15天，每天跟430和KSSSS等著名职业选手，针对性练习，虽然在比赛时有很多机会，但是还是输给了当时已经成名的NV、LGD和EHOME战队，若风非常遗憾的再一次回到了原点。

　　输了2010年WCG之后，若风开始懂事了，一方面他有2年的时间没有赚钱，也没有上学，这让他对家人很愧疚，另一方面他对自己在DOTA方面很失望，有些怀疑自己可能不适合走这条路，于是他答应母亲去上学，希望这样可以给他们一些安慰，于是他重新拿起书本开始补习，考上了湖南传媒大学的专科，开始了另一段生活。

　　开明的父母，未完的约定

　　若风的母亲在看到若风对待电竞梦想的认真态度后，跟若风定下了一个约定，如果大学期间若风想去打职业，父母绝不会反对。2010年9月若风在学校里第一次接触到了英雄联盟这款游戏，开始了另一次的征程。

　　当时若风和Ayaya是在长沙的Dream战队，他也是在那个时候认识的现在WE的教练Joker，我们为了参加了由腾讯举办的TGA大奖赛，每天在网上训练，一个月打到国服前30名，最后在线上赢了所有的队伍，还记得最后一场比赛的那天晚上，我们全寝室的人都在看他打比赛，他和队友不断的用语音软件沟通交流，赢的时候已经是快凌晨了，若风当时大吼了一声，估计全楼都可以听到，真是赢的非常痛快。可惜后来，TGA线下总决赛的时候，他们输给了当时全国唯一的一只英雄联盟职业战队EHOME，不过这也让他们受到了关注。

　　受邀加入电竞豪门WE

　　若风和队友的出色表现让WE俱乐部经理裴乐KING和周豪ZAX十分看好，在经过多次沟通后，若风和队友毅然决定远赴上海，加入这个世界著名的电子竞技俱乐部TeamWE，成为一名英雄联盟项目的职业选手。

　　加入WE后的生活并不轻松，当时国内还没有英雄联盟的比赛，若风和队友们为了提高水平，顶着200PING的网络延迟，每天晚上12点到第二天中午12点，在美国的服务器上跟国外的队伍交手，因为当时比赛少，所以几千块奖金的比赛都有很多高水平的战队来争夺，WE战队那段日子过的痛并快乐着。

　　由于WE稳定的训练环境和多年的管理经验，若风和队友的成绩开始有了突飞猛进的提高，终于在2011年10月1日，在广州举行的IEM世界大赛上，WE.LOL分队一举击败当时的世界强队美国CLG，欧洲AAA，新加波Flash战队，获得第一个世界冠军，赢得1万5千美元的奖金，并在11月拿到TGC大奖赛的冠军，赢得20万奖金，到2012年更捷报频传，连续WGT、NGF、TGA2012的多个大奖，并且在8月去韩国参加为期一个月的OGN大赛。

　　若风作为WE英雄联盟战队的队长，也被英雄联盟的出品公司美国Riot公司邀请到美国代表中国职业选手参加年会。并在2012年代表中国参加在美国洛杉矶英雄联盟母公司Riot举办的第二赛季世界总决赛S2。

　　父爱重于山

　　若风在2011年WCG中国区总决赛输给IG后，他自己很失望，因为WCG对他有着非常重要的意义，他很想赢，但是输了，哭了30分钟，回到酒店，第一次主动打电话给他父亲，他自己也很奇怪当时为什么会打给父亲，因为他父亲在他小时候一直在念书，所以聚少离多，有一段时间对父亲的教导非常抵触，觉得父亲说什么都是罗嗦，后来父亲对他也很无奈了，就什么都不说了，于是有很长一段时间，父子之间都没什么交流。

　　那天若风打给父亲，哭着跟父亲说对不起，觉得这么多年来，让父母失望了，但是让他很意外的是他父亲竟然没有责备他，并且还不断的鼓励他，肯定了他现在做的事情，一瞬间很多年的隔阂都不见了，他们父子现在变成很好的朋友，会经常打电话聊天，若风很开心。
