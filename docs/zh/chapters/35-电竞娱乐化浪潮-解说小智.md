# 三十五 电竞娱乐化浪潮 解说小智

> 首发于知乎专栏（2014-10-17）原文链接：https://zhuanlan.zhihu.com/p/19869029

中国电子竞技幕后史 三十五

电竞娱乐化浪潮中的解说小智

文/BBKinG

　　现在提起解说小智，有人会好奇他从不公开的长相，有人会好奇他做电竞解说赚来的千万级身家，我在没有采访他之前，也很好奇他这种很接地气的“智式幽默”是怎么形成的？甚至一度猜测他是不是天桥艺人出身。

　　而当他真的带着司机，开着豪车来葫芦岛车站接我，并跟我聊了两天后。

　　我才发现，这个每星期在LOL游戏视频里，变着法子给大家讲笑话，逗大伙开心的解说，竟然是个言语不多，但对历史有着浓厚兴趣的官三代。

　　小智的爷爷，以前曾是主管当地经济的地方官员，父亲曾是银行行长，后来下海开厂经商，一个舅舅是镇上数一数二的商人，另一个舅舅是公安局局长，整个家族根基庞大，简直就是威震一方的土皇帝。

　　按照剧本，小智在这样一个环境下长大，怎么都不应该是现在这个样子吧？

　　的确，至少在2009年之前，小智的人生是另一个样子。

　　杨丰智，1989年1月6日出生于辽宁锦州，与一般的孩子不同，他有个天生的缺陷，大舌头，这让他的童年备受困扰。

　　从小被周围人嘲笑是一种很残酷的经历，这让他的性格变得敏感、倔犟、暴躁、好斗，特别是爱打架，最终在很多年后，给他惹了大祸。

　　还好，面对人生，小智不认命，从小学开始，他就逼着自己练习如何跟别人说话。

　　于是，在自习课上给大家讲笑话，便成了一个很不错的机会。

　　一开始，小智一张嘴，对于同学们来说就是个笑话，他并没放弃，一边不断收集生活中的幽默素材做积累，一边学会了自嘲。

　　就像《权利的游戏》里，侏儒男主角小恶魔说的：永远不要忘记自己是什么人，因为这个世界不会忘记。要化阻力为助力，如此一来才没有弱点。用它来武装自己，就没人可以用它来伤害你。

　　小智说，自己早期的幽默其实很简单肤浅，就是单纯肢体上的幽默。比如，上台的时候会假装笨拙的摔一跤，引起哄堂大笑。

　　2年后，小智不但肢体语言上应用自如，还摸索出了一套起承转合更加流畅的语言技巧，通过主动放大自己的缺点，利用自嘲制造荒诞的反差效果，这也就是现在“智式幽默”解说风格的雏形。

　　其实，我很想在这里写：从此，他和同学们过上了和谐共处的生活。

　　但是，生活这个导演，似乎不甘心让小智的人生如此平淡。

　　小智说，孔二狗写的《东北往事黑道风云20年》是很真实的，东北人的好勇斗狠，成长环境恶劣，这些都是他亲身经历的。

　　小智曾经在开学第一天去学校的路上被打劫了5次，有时在网吧里玩游戏嗨了撇句话，都能惹出一场地区性帮派群殴。

　　在这个环境中长大，想不被人欺负，要么你服软，要么你比他能打。

　　小智说，我想服软来着。

　　抢劫的第一次把砖头举起来的时候，他机智的掏钱了，第二次，他还是主动掏钱了，后来，有人组队来他班上要钱的时候，他怒了。

　　那一场架打的…

　　小智只记得自己拿着砖头，往领头的那人头上重复的砸，砖头渣滓在乱飞，至于周围有多少个人在打自己，他脸上和身上直到现在还能看出的几道伤疤是怎么留下的，他已经忘记了。

　　他开始明白，在那个环境里，想不被欺负，不被嘲笑，就只能反抗，要比别人更强。

　　后来，他被学校开除了，因为之后无数次的反抗，无数次的打架。

　　初中成了他最后的学历。

　　被开除后，小智的父亲非常生气，让他继续上学，但是他死活不去了，学校给了他太多不好的回忆。

　　父亲气也上来了，扔下一句，有本事你一直待家里别出去。

　　小智倔脾气也上来了，说，待就待。

　　没想到，这一待就是一年。

　　每天父亲出门前反锁房门，拉掉电闸，只扔给他一堆书，他就在家里看书，没有电玩电脑，看电视，甚至厕所灯都不亮。

　　小智说，就是为了憋一口气，他们说我待不住，我就是要待给他们看。

　　在这一年里，他把家里的书都看完了，罗马帝国衰亡史，春秋战国，唐宋诗词，明史，清代诗歌等等，这一年真是对他影响很大，以至于他后来喜欢的东西都和同年龄人不太一样，比如，别人散步，带耳机听流行歌曲，而他喜欢把手机插在上衣口袋里，用公放听太平歌词，活脱脱一个小老头。

　　一年后，小智的母亲实在看不下去了，这要把人憋出病来怎么办？

　　这下放虎归山了。

　　解放后的小智从此在一个很破旧的小网吧里，过上了据他说是人生中最快乐的时光。

　　传奇，CS，WAR3，澄海3C，只要是能跟别人一起玩的，他就没日没夜的玩，在网吧里不但认识了很多朋友，还有中学时认识的好友陪他，碧哥，也就是现在他每个视频里都会提到的重要人物，这是个真人，是小智当年唯一的一个朋友，后来碧哥去了别的地方上学，小智十分落寂。

　　最重要的是，那个网吧的老板据说年轻的时候也是当地一霸，附近的小流氓不敢进来惹事，他很照顾小智，这让小智第一次有了一种找到组织的温暖。

　　日子就这样过着，虽然小智很开心，但是他父亲却很痛苦。

　　小智说，老东北人的思想里还是很传统的，认为人总要有个班上，有个养家糊口的工作，但是小智这个样子，他父亲也犟不过他，于是，只好改变自己的人生计划，放弃公职下海经商，他父亲的想法很简单，如果能挣些钱放那，随便小智后半生爱干嘛干嘛。

　　真是可怜天下父母心啊。

　　没想到，这一下海，生意还真好。

　　小智从官三代，直接变成富二代了。

　　从2005年开始，父亲为了培养将来的公司接班人小智，特意带着他参加工厂里的会议，还带他去做采购，学习公司运作的各个层面。

　　按照这个剧本往下演，我觉这篇文章应该是一个乡镇企业家的前传。

　　不过，老天爷这个戏霸，又跳出来改戏了。

　　2007年的一次化学事故。

　　厂里的一个模具工人把一个化学药品在没有密闭的情况下，忘在了宿舍里，小智那天下午忙完，就进去睡觉了…

　　过程我们就不说。

　　抢救过来后，小智身上开了一个40厘米的刀口，彻底告别实习岗位，被送回城里养伤。

　　告别了乡镇企业家培训班，对父亲来说可能是件很遗憾的事情，但是对小智来说，却歪打误撞成就了另一个开始。

　　电子竞技解说

　　休养期间不能工作，但是却能玩电脑的小智，疯狂迷恋上了CH3C（澄海3C，魔兽争霸3 RPG地图之一），2008年1月，他常看的一个解说视频作者发了个邮件。

　　在邮件里，小智建议这个解说在视频中加入幽默搞笑的元素，这样会更吸引人。

　　也不知道是幸运，还是不幸。

　　这个当红的解说很罕见的回复了这个邮件，不过，却在里边很不客气的说了句：我就是不喜欢这种风格，你行你来啊！

　　小智的犟脾气当场就犯了，来就来！

　　很快，按“智式幽默” 做的第一期澄海3C的视频发在了土豆上，几乎是瞬间就有了5万的点击，比那个解说还高，这在2008年，可是相当不错的成绩了。

　　观众也表现出了巨大的认同感，这让小智仿佛一下子找到了人生方向，他以一种狂热的状态投入到这个工作中，开始以每天一期的速度上传视频。做过视频的人知道，这么大的工作量，是很耗费心力的。

　　很快，点击量也有了质的飞跃，最高一期有70万的点击。

　　这一年，不到20岁的小智，生活也很滋润。

　　家里每月给他一万块的零花钱，这足够他和他的女朋友衣食无忧，还能到处旅游。

　　物质精神双丰收，不过按照小智这个人生剧本的尿性，不用我说，大家也猜到了，转折又一次说来就来。

　　2009年，小智的父亲突然被同一个家族的合伙人阴了一把，经营正火的工厂一夜之间跟他没关系了，而为了建厂贷的很多款，却落到了他的头上。

　　而小智，一个正往文艺道路上狂奔的无忧少年，瞬间变成倒卷衣袖为口忙的屌丝。

　　没钱，没工作，没学历，家里还有巨额的债要还。

　　这个从天上到泥里的过程，甚至没给小智什么消化的时间。

　　下个月一分钱都没了，靠什么吃饭？

　　经过短暂权衡，小智拿着父母给的最后一个月零花钱，在当地一个商场租了一个连水电费全包进去一年才4000块的店铺，剩下的钱打算跟着亲戚去学卖衣服。

　　想卖货，先要学会如何进货。

　　之后的一年里，小智每个月至少有2天是这样度过的。

　　晚上7点，和女朋友一起坐绿皮火车，从葫芦岛去北京，因为不敢买动车的票，凌晨一两点才到北京火车站，下了火车，先飞奔去通宵窗口买返程票，为了有座位放包。

　　买了票之后，去二楼的麦当劳点一份薯条，每次女朋友都让他先吃，他吃一半剩给女朋友，女朋友吃一半，再推给他。

　　吃完东西就往北京动物园赶，那里有一个服装批发集市，每天只从早上5点到12点存在，所以要4点前就去占好位置。5点不到，几千个从各地来抢货的人，就会在漆黑一片的大门前静静的等待着，那种压抑的感觉，让小智至今难忘。

　　一夜未眠加上一早上夺食一般的扫货，中午12点，小智和女朋友一人扛一个大包走了出来，直奔地铁站，甚至舍不得坐三轮车。

　　坐上地铁后，还得在北京地下绕个几圈，因为下午回去的火车是3点的，火车站人太多了，去早了连个下脚的地方都没有，地铁里虽然挤，但还能坐会。

　　小智说，那个时候最大的梦想是有三万块钱，能买个二手普桑去进货。

　　如今回首，小智很感谢那段日子，每次赶车，绿皮车厢里都挤满了形形色色的人群，有去北京上访的，有幻想一夜暴富的女性，也有满是纹身的男性，他开始接触到这个真实的社会，也接触到了真实的自己。

　　经过这些大起大落的磨练，小智放下了曾经的清高倔傲，学会向生活妥协，学会了如何从一笔一笔小生意里踏实的活下去，这些都给他后来做淘宝店铺提供了大量经验。

　　这一年他才20岁，人生已经上下翻转多次。

　　2010年，在几个喜欢澄海3C的朋友支持下，小智把服装店盘了出去，又重新开始做解说视频，但是他发现此时的CH3C，已经在DOTA崛起中没落了，面对困境，他并没有放弃，反而觉得应该做更多的比赛来振兴这个项目。

　　转身就去搞比赛了。

　　为了给比赛拉赞助，小智去北京陪富二代们喝酒，不善于拉赞助的小智被富二代们逗的团团转，他们开出喝一杯酒给一千块，为了能做成比赛，本来不怎么喝酒的小智玩命了，他只记得那天带了6万多块走的，后来吐了几天。

　　之后比赛执行阶段，小智亲自组织，亲自通知队伍，大量的心血花在CH3C上，但是结果却适得其反，玩的人越来越少，还因为严格执行规则，得罪了不少人。

　　辗转2年后，徒劳无功的他看到了袁崇焕写的那首诗，大哭了一场，彻底放弃了自己最爱的CH3C。

　　一生事业总成空，半世功名在梦中。死后不愁无勇将，忠魂依旧守辽东。

　　2012年，经济崩溃，精神也几近崩溃的小智，一度心灰意冷，打算彻底离开这个圈子。

　　最终在北京一位姓孙的朋友的几番建议之下，决定重新出山做英雄联盟的视频，这个朋友十分看好小智幽默的解说风格，他认为小智会在这个新电竞项目中大获成功。

　　于是，小智拿出自己的全部积蓄70元，对，你没看错，这就是他2012年的起点，加上孙和另一个姓姚的朋友借给他的3000块钱，去买了一台2600块钱的新电脑，带着有些悲壮的情怀，以及他所有的人生经验和感悟，踏上了一段新的征程。

　　如今，小智一期视频在各大平台的点击量将近2000万，他的淘宝店铺涉及服装、外设、零食、鞋子等等，现金流水每年翻番，2014年秋天，他以天价签约斗鱼直播，同时开辟视频点播和直播互动两条路线，将有更多的人通过他的解说，认识小智，认识中国电子竞技。

编者按：

　　2010年之后，中国电子竞技文化在80后进入30岁，90后步入社会等大时代背景下，开始从校园往社会扩散。

　　电竞人群在急速扩大，市场需求也变得更加广泛，大众在工作和生活压力之外，需要有更加休闲的娱乐方式，满足大家放松的需求，这便产生了电子竞技娱乐化的浪潮。

　　面对浪潮，有人欣喜，有人反对，而我却觉得很幸运，因为在这个时代，可以有如此多样性的人和文化可以同时迸发出来，还能有如此多的变现手段来进行商业化融合，这让电子竞技未来几乎可以产生无限的可能性。

　　解说小智，是这股时代浪潮中的一个缩影。
