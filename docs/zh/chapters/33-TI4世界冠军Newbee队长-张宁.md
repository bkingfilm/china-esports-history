# 三十三 TI4世界冠军Newbee队长 张宁

中国电子竞技幕后史三十三 TI4世界冠军Newbee队长 张宁

文/BBKinG

　　张宁说，TI4总决赛赢的时候整个大脑一片空白，朋友说他颁奖典礼时整个人就像在梦游一样，他说的确以为自己是在做梦，以前都是看别人拿冠军，幻想自己是冠军队，如今自己真的拿冠军了，直到隔天睡醒了才相信这不是个梦。

　　第二天，全世界的新闻媒体，都报道了张宁所在的Newbee战队在美国西雅图拿到TI4 DOTA2世界总冠军的新闻，而所有几乎报道都把关注点集中在了那3000多万人民币的奖金上。

　　既然这么多人说钱的事了，我们就说说人的事吧！

　　张宁，阴历1989年12月14日生于湖南邵阳，网上昵称xiao8，这个名字起源于他玩《梦幻西游》时用的网名“小小”，打DOTA时在VS平台注册名字，发现xiao1-7都被注册了，于是就有了xiao8。

　　张宁的父亲是一位建筑承包商，母亲是家庭主妇，有一个大5岁的亲哥哥。

　　从理论上说，这本应是一个富二代的故事，因为张宁从小家境还不错，母亲现在每个月的主要工作是到处收房租，但是，用张宁的话说，他父母继承了中国一项古老的传统教育美德——抠门，哦不，是严苛！于是，这个故事就走向了地主和杨白劳的节奏。

　　其实上小学时，张宁的成绩是很好的，基本上都是全校第一，人见人爱，每个老师都在夸，父母也很自豪，于是就答应他：如果中考也能考到全校第一，就给买台电脑。

　　这对张宁真是很大的吸引，要知道之前他去网吧玩，不但贵，还会被他爸揪出来打。

　　结果是全校第一是拿到了，电脑的事被忘了个干净，从小被打到大的张宁也不敢找强势的父母问个原因，就这么沮丧的进了中学，也走向了叛逆。

　　初中之后的张宁变得非常调皮，还记得你上初中的时候，老师会把班里最难管的人单独安排坐在讲台旁边吗？不要嘲笑他，那小子将来可能会拿世界冠军。

　　张宁以前跟父母的关系很不好，父亲很强势，经常不是打就是骂，母亲虽然刀子嘴豆腐心，但也让张宁活的很压抑。往往小时候越被压，青春期就越叛逆。

　　张宁就这样走上了小混混的道路，逃学玩游戏，到处惹事，每个星期都会被请家长，2006年高二的时候就休学了，跟三四个朋友一起开了个《梦幻西游》的游戏代练工作室，每个月除掉开支，竟然能分到手一万多块，不少钱了，但这对一个16岁孩子来说，反而让他更迷茫，因为这个年纪没什么需求，也不知道怎么花，那赚这么多钱干嘛？

　　于是，当父亲问他要不要继续读书时，他觉得出去长长见识也不错，就去了广东中山读计算机软件。

　　离家远了，反而学习好了，他很喜欢计算机编程，但很寂寞，因为语言不通再加上人生地不熟，周围一个朋友都没有，他只好每天在网吧里跟邵阳老家的朋友们语音聊天，这个时候，湖南的朋友圈里开始流行一款叫《真三国无双》的游戏。

　　说起这个游戏，真是中国DOTA职业选手的发源地，后来相当多的著名电竞职业选手，都是当初玩真三时的对手和队友。张宁在这里认识了后来对他人生轨迹影响很大的一个对手——诸司马技 ZSMJ 。

　　2009年，当还在打真三的张宁，看到早已转DOTA的ZSMJ跟着伍声（2009）的LGD战队登顶吉隆坡SMM世界冠军时，他开始转型DOTA，有半年时间都在边打真三，边练DOTA。

　　毕竟真三时期不俗的成绩，也让他的转型很快受到了重视，zsmj第一个递来了橄榄枝，跟他说有机会去LGD，当时09已经退役，队里缺个主力。

　　有世界冠军光环的队伍，这对张宁来说是很有吸引力的，于是开始专心在VS平台的一房里练了一个月的DOTA 。

　　张宁此时给自己立了个目标，要打职业，要拿世界冠军！

　　可惜的是，后来LGD原来的队员小池突然归队了，没了张宁的位置，这个事情只好就此作罢。

　　不过，张宁的运气没断，另一个著名的DOTA职业选手xB找到张宁，邀请他加入有LongDD等人的国内强队成都CH战队，机票都帮他买好了，他也从邵阳到长沙准备坐飞机去成都。

　　就在这时，长沙的好友YAO极力劝说张宁留下，因为长沙要组一个新队，由当地一个资金雄厚的老板出钱，队伍里又全是本地朋友，最关键是离家近，这些让看重兄弟朋友的张宁最终选择留下，组成了后来的Dream战队。

　　当时Dream战队的队友有：YAO、DD、LI、老鼠、张宁。

　　1500块一个人，不包吃住，但是张宁很开心，一方面是因为终于又跟朋友们聚在一起了，另一方面也是因为刚打职业，觉得一切都很新鲜。

　　可那一年的成绩并不算理想，印象最深的是，刚建队2个星期就被带去泰国参加一个DOTA比赛，结果第一轮16进8就被一只完全没听说过的队伍淘汰了，晚上大家在酒店里大哭，都没有心情出去玩。

　　国内比赛也是拿了很多的第二第三，因为头上有个如日中天的战队压着，这个战队叫Ehome，2010年的Ehome战队，有着820、DC、Burning、KinGJ等大神，可真是人挡杀人，佛挡杀佛。虽然张宁他们并没有拿到第一，但是他们毕竟年龄小还是只新队，大家都干劲十足，一直在相互鼓励。

　　一切似乎都在向好的方向发展着，世界冠军不是太远，只要再给张宁和他的队友们一些时间。

　　不过，历史是残酷的，2011年春节前，Dream突然接到通知要去北京参加一个比赛，由于正值春运期间，火车票买不到座位，几个人就这样一路站到北京，在去之前领队也突然说了一句很莫名其妙的话，说如果这次比赛拿不到前三，这个月工资就不发了。

　　在又累又莫名的情况下，Dream拿到了第三，但是那个月的工资还是没发，而且回家过完年后，领队也联系不上了，这个队伍就这么莫名其妙的散了，不过，这在当时的电竞圈也是比较正常的事情，很多战队和俱乐部都没有公司化，队员之间连合同都没有，说没就没了。

　　不过，是金子总有人要，没多久，同是湖南老乡的DOTA职业选手zhou和另一个职业选手狗哥两人组建了CCM战队，拉到了一个北京老板的投资，愿意给张宁5000一个月，包住，向往打职业、拿世界冠军的张宁同意了，瞒着家人说是去北京工作，就此踏上了家乡之外的新征程。

　　此时CCM的阵容是：430、DDC、zhou、狗哥、张宁

　　可是，美梦只做了3个月，3个月后，老板联系不上了，虽然之后也拿了几个冠军，还有个叫IPDC的马来西亚比赛，可主办方直接卷了奖金跑了，CCM陷入了困境，大家也没什么心情训练了，每天都是各玩各的，张宁对战队的不满也越来越深。

　　在2011年8月，WCG中国区总决赛的那天晚上，也就是幕后史前一章讲到的笑笑他们作为CCM的LOL分队夺冠的那天晚上。

　　万达公司董事之一的王思聪跟所有CCM DOTA分队的人一起面谈，宣布了正式收购CCM的消息，并将战队名改为IG，补发了之前欠的工资，承诺工资翻倍，还有笔签字费，这意味着张宁的工资将涨到一万块，只要签合同，就马上有一大笔现金到手。

　　但是张宁还是决定要走，并不是跟钱过不去，而是他觉得这个战队在之前危难时表现出的涣散，让他很心寒，特别是好友DDC的一句话更让他坚定了离开的想法，DDC说：要那么多钱干什么？一起打的开心就好。

　　随后张宁和DDC接受老友zsmj的邀请，去了4个湖南老乡加一个澳门人的LGD战队。阵容如下：张宁、DD、DDC、YAO、ZSMJ

　　此时的LGD负责人是DOTA圈著名的女强人RURU，给张宁开的工资是4000块包吃住，训练在天津的一个复式楼里，虽然工资比IG开的少，但张宁很开心，一方面是又跟老友聚在一起了，另一方面是因为他发现训练基地竟然有专门打扫卫生和做饭的阿姨，他第一次觉得打职业是件幸福的事情。

　　可是有一段时间，LGD战队的成绩很不好，在WDC止步8强的那天晚上，ZSMJ红着眼找到张宁，说自己心思不在这里了，打算休息一段时间。

　　WDC后，对张宁职业生涯影响深远的ZSMJ退役了，可战队还是要往前走，LGD通过俱乐部转会协议买来了原WE选手Sylar，战队成绩开始有了起色。

　　之后的事情就一帆风顺了，LGD有规定，奖金都给选手，俱乐部不分，只要比赛能进决赛就给加工资，一年后所有队员都加到了一万多。国内电竞环境也越来越好，张宁也更加出名。

　　2014年，本来打算退役的张宁在老朋友王玥的劝说下，加入新成立的Newbee战队，跟香蕉、狗哥、HAO、MU再次出发冲击自己的世界冠军梦。

　　TI4 圆梦之旅

　　张宁说，2012年，自己所在的LGD在西雅图打TI2世界总决赛的时候，因为前边太顺了，十七连胜进了胜者组决赛，以为冠军就在眼前，结果那场比赛莫名其妙的整个队伍出现各种沟通失误和低级错误，导致输给了自己，输了之后心态也坏掉了，最后只拿到第三名，十分可惜。

　　2014年TI4世界总决赛，正好是反过来，Newbee小组赛一直输，最后是打加时赛硬挤进的前8，像是从悬崖边上捡回来一条命一样，所以后面打的格外珍惜，但是也压力巨大，每天饭都没胃口吃，晚上睡在床上都要想几个小时明天怎么打才睡的着，反而是这种压力让整个队伍都发挥出了潜力。

　　惊险的总决赛

　　在总决赛前，有2天的休息时间，可这两天Newbee几乎没怎么吃饭睡觉，都在看对手比赛，研究对手，生怕错过任何一个细节，以至于都没有摸过键盘鼠标，结果决赛第一场打VG，因为手生也因为压力很大，就输了，还好在输了第一场后，大家觉得不是实力问题，而是状态问题，只要能调整过来就可以赢，最终以3：1拿下了TI4的世界总冠军。

　　追女孩和争世界冠军一样要用心

　　如今的张宁真是升职加薪，当上世界冠军，迎娶白富美，走上人生的巅峰。只不过，是先迎娶了百富美。

　　成家立业，这个顺序有时候还是蛮有道理的。

　　2011年，张宁在黎明前最黑暗的时候，认识了之后的上海老婆赵洁，当时的赵洁已经是名气在外的小明星了，而张宁则是一个生活都不算稳定的游戏屌丝，赵洁刚开始也不玩游戏，是来WCG帮朋友来要签名的，连DOTA都看不懂。

　　表白被拒、异地恋、上海丈母娘不同意，这几乎就是普通人眼里的三座大山啊！

　　可是，性格内向、不太说话的张宁从第一眼就认定这个很开朗、喜欢说话的女孩是自己的梦中人了，作为摩羯座，要么不出手，要么锁定目标就会很执着。

　　大山被一一克服，丈母娘甚至也开始通宵看他比赛，2014年4月，两人领证结婚，注意！这一切都发生在3000万奖金之前，所以，张宁的案例告诉我们，追女孩还是要用心，别的都是借口。

　　就像追逐梦想一样...

　　TI4世界总冠军Newbee队长 张宁
