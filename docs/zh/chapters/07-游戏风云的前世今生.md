# 七 游戏风云的前世今生

> 首发于知乎专栏（2013-11-22）原文链接：https://zhuanlan.zhihu.com/p/19620732

中国电子竞技幕后史七 游戏风云的前世今生 文/BBKinG

　　2006年，在完成第三届StarsWar后，我正式调入当时还叫“游戏风云GamesTV”的上海文广互动游戏风云频道（现在已经成立了独立运作的游戏风云有限公司），接触的第一个项目就是第一届G联赛（G·League）的筹备。

　　当我在整理资料做PPT时，发现G联赛还没有中文名，于是拟了一个“中国电子竞技电视联赛”的名字拿去找领导们审核，会审的时候，频道总编室主任柏玉海老师提了个建议，“中国”这个词只能由国家级体育部门用，我们最好用“全国”，大家都觉得挺好，于是，G联赛的中文名“全国电子竞技电视联赛”就这样被定下来了。

　　斗转星移，7年后……

　　2013年3月9日早上10点，G联赛第二赛季总决赛将在上海“赛德斯-奔驰文化中心”举行。

　　这是一个8000人的场地，也是上海现在最炙手可热的演唱会举办地，可能没做过大型活动的人不太了解这背后意味着什么。

　　G联赛放在奔驰文化中心，一方面意味着各方面的执行难度将大幅度上升，4000人到8000人，不是简单的乘以2的关系，它的复杂度增长是几何式的，小到后勤支持和售票渠道，大到项目审批和活动成本控制都将面临前所未有的考验，游戏风云的团队执行力和配合熟练度将在这场活动中被全方位考核。另一方面，这意味着电子竞技开始更加商业化，高额的场地费用，地铁和户外广告费用，这些都是在对电子竞技之外的人们发出声音，电子竞技也可以在这样的场地里办演出了，这是一次质的飞跃。

　　无论结果如何，作为曾经的游戏风云一员，我为这个团队在这些年的坚持和探索而骄傲，并将本文献给所有为中国游戏视频节目发展作出贡献的电视人。

　　命中注定的游戏缘

　　2004年，喜欢FPS游戏的吴强（GamesTV前总经理）、喜欢足球游戏的张哲希（现游戏风云副总经理）、喜欢主机游戏的张锐（现游戏风云制片人），以及已经退出游戏圈的文圣可等人在一个咖啡馆里一起组建了GamesTV。

　　还记得上一章讲StarsWar时，提到过2005年IGE中国总经理许云波被派回中国，负责组建上海代表处的事吗？IGE美国总部为了获得话语权改变虚拟物品交易商的形象，在美国收购了一些游戏传媒公司，效果很不错，于是就打算在中国复制这个路线。

　　刚好文圣可是许云波的同学，这个线就搭上了，这就是命，咱不得不信啊。

　　于是，2005年夏初IGE收购了GamesTV，而GamesTV就是游戏风云的前身。GamesTV最初的想法也比较简单，就是打算做游戏方面的视频节目，然后看看有没有机会能销售广告。之所以从电子竞技开始做，因为当时CS还在火，WAR3刚起步，电子竞技相比当时的网络游戏是一个目标比较明确的切入点。

　　2005年6月左右，IGE公司搬到了位于威海路文新报业大厦的办公室，GamesTV和我所在的[http://Raplays.net](http://link.zhihu.com/?target=http%3A//raplays.net/)项目组随后也全部搬了进去，两个项目组直线距离不超过10米，我也就是在那个时候认识了妖魔、吴强和文圣可等人，从此大家开始了同事的生活。

　　由于GamesTV在起步阶段打算从电子竞技入手，所以寻找“能解说能上镜”的电竞人才就成了第一要务。

　　最初是让[http://Replays.net](http://link.zhihu.com/?target=http%3A//replays.net/)的创始人ZAX担任上镜解说，专门做War3的教学视频。但是ZAX本身有RN和StarsWar的工作不能分身，需要物色一位更合适的人选。

　　于是，BBC在妖魔的介绍下进入了GamesTV并很快上手，从此开始与主播台结下不解情缘。职业选手MagicYang也在那个时候被拉入的，后来经典的黄金搭档BBC和老杨组合也从那时开始成型。

　　GamesTV最初的设备相当简单，录影棚就搭在公司里一个10平米不到的空房间里，聚光伞是工作人员自己找人焊的，摄像机大多只是家用级的。

　　不过那段时间发展的速度蛮快的，在很短的时间里加入了一大批强人。

　　张楚齐，40岁不到的老顽童，我们一般尊称他为老张，特种部队伞兵出身，高大威猛红脸关公式的武汉人，复员后在电视台里摸爬滚打十几年，标准的老电视人习惯，脾气火爆，技术高超，我们都被他骂过，BBC刚做主持人时还被骂哭过，但是都很服气。

　　StarsWar第二届到第四届都是他做导播，我做现场导演，我在现场也没少挨骂，刚开始有点受不了，后来发觉他只是就事论事，事做好了就会端个茶杯笑的跟孩子一样，有他在时，我们每个人都要提十二分的精神做事，他就像一个皮鞭，我很庆幸那时候团队里有一个这样的人。

　　可是，老张也毁在他这个脾气上，急起来谁也不认，领导嘉宾都不给面子，这在电视台里必然会碰钉子，2006年他离开了游戏风云，现在进了出版行业。去年老张来上海一起吃饭，个性依旧，结婚有了儿子，人似乎平和了不少。

　　另外2个强人，可能上海的电视观众会比较熟悉。

　　2002年，上海生活时尚频道开播了一款后来很受大家欢迎的游戏节目《游点疯狂》，相信很多人都记得，但是后来因为2004年广电的游戏节目禁令而下线。几经辗转，这个节目的总策划黄昊和主持人张锐在2006年进入GamesTV。

　　黄昊被我们尊称为黄老大，导演大胡子个性十足，他是一个思路非常清晰的制作人，电视制作经验丰富，2006年，他在一个活动现场，看着忙忙碌碌的我，不经意的对我说的一句“在现场，导演应该是没事的”，让我对如何带团队做项目茅塞顿开。

　　张锐是上海戏剧学院表演系出身，播音十分正统专业，我刚写好的稿子，他可以拿起来就配音，而且可以不带停顿的自动改善语句不通顺的地方，这经常让我感叹隔行如隔山，他本人酷爱XBOX360等主机游戏，接触很早涉猎精深，在主机游戏方面是绝对的权威。

　　黄昊和张锐的加入不但带入了很多电视制作的编导后期等人才，还带入了电视台的节目制作思路和标准，这让GamesTV在节目种类和质量上上了一个大台阶，开始不但有电子竞技比赛，还有了单机游戏和游戏新闻资讯类节目。

　　小荣117（现游戏风云赛事总监）在2005年从AS战队退役后先是进了浩方负责赛事，后来在2006年被拉入游戏风云，小荣从职业选手的角度解说FPS比赛让人眼前一亮。

　　小荣是个十分聪明的上海人，据我个人的观察，在大城市长大的人往往善于与人相处，小荣很符合这一点，另一点是他会接住领导扔下来的工作，并且努力完成。要知道这并不容易，因为有些工作是十分枯燥但是又不得不做的。

　　没有领导会不喜欢这样能分压的下属，所以即使在2008年游戏风云分家时，小荣也是第一批被点名带到了文广的。

　　同是摩羯座，这些年我能看出来小荣在隐忍，可是代价也是相当大，上个月我去游戏风云看他们，小荣头发都少了很多，心态也老了很多，让我十分感慨当年的木村拓哉怎么变黄渤了。

　　还记得做CS MTV系列的甜咖啡吗？

　　他后来也在吴强的劝说下从北京来到上海加入GamesTV，他的人生经历比较坎坷，从小形成独特的观察能力，他会用很不一样的视角去发现我们身边习以为常的事情，2008年他回到了北京，现在在完美国际工作。

　　初露锋芒

　　GamesTV的第一次发力是2005年11月在新加坡举行的WCG世界总决赛，在这之前张哲希花了大量时间和精力跟WCG组委会沟通，最终拿到了免费的赛事直播权，如果做成功，这将是绝大多数中国电子竞技爱好者第一次同步看到WCG现场画面，所以IGE也很支持，派出了GamesTV前方报道组，妖魔等几个同事前往新加坡做现场直播保障和出镜采访。

　　我还记得当时因为拍好的视频采访太大来不及传回来，为了抢时间，就用了电话连线录音的方法。有次采访前，我刚好在他们办公室里，听到妖魔很疲惫的声音在跟吴强沟通等会的连线，他们已经连续工作了几天了还不能休息，因为WCG就是那么几天，第一手的信息有多重要相信大家都了解。

　　辛勤的付出，回报也是非常可观的，GamesTV在对WCG的直播中表现出色，关注度大幅度上升，特别是那一年SKY拿了中国WAR3的第一个WCG世界冠军，而且是相当有份量的世界冠军，全中国的电子竞技爱好者都沸腾了。

　　也正是因为有这么好的效果，到了第二年，2006年WCG世界总决赛的转播权就有人争了，当时的NEOTV出了10万美金，拿走了本来免费给GamesTV的转播权，因为WCG组委会觉得2005年与GamesTV合作蛮愉快的。

　　也是正因为这件事情，NEOTV跟游戏风云埋下了怨恨，这是后面要讲的事情，我们这里先不展开。

　　下面要说下从2005年到2006年GamesTV存在的问题了

　　回首GamesTV的这2年，除了StarsWar2的直播和WCG2005的直播比较出众外，事情做了很多，可亮点并不多，为什么？我觉得有几个原因。

　　1、公司思路不清晰

　　这要从IGE和GamesTV当时的高层说起，先插个题外话，我和许云波现在的私人关系很好，但是有时候一起喝高了当面骂骂当年的他，他也不生气，因为我们都明白如果换了我是那时候他，我可能也不知道该怎么做。

　　当时的IGE每天的纯利润是30万美金，外界对公司估值6亿美金，人并不多但是非常赚钱，而且从2006年左右那个网络游戏市场的火爆程度上看，可以说是钱途不可限量，于是，对于IGE的老大们来说，下面的资讯网站不需要考虑赚钱的事情，把节目做好就可以了。

　　可是，如何把节目做到更好？上面没有压力没有方向，时间长了，下面的人自然很茫然。

　　2、当时没有固定的播出平台和节目量，收视率不高

　　当时网络游戏运营商每年在17173等网络游戏门户网站上的广告投入是非常大的，但是在视频媒体方面的投入却很小，一方面是当时玩游戏看视频的习惯和需求量并不大，另一方面是视频播放平台还不成熟，优酷这样的视频网站到2006年6月才出来，当时网络情况也不如现在，用户不像现在看视频这么方便流畅。

　　到了2006年，另一个麻烦又来了，这就是《信息网络传播视听节目许可证》和《广播电视节目制作经营许可证》问题。

　　其实这两个证的政策2004年就发布了，但是那几年互联网的发展本身就比较乱，大家都在试探和观望，也没见谁赚到钱做了出头鸟，所以广电虽然出了政策却也没有一棒子打死。

　　可是，这就像一把悬在头顶的利剑，什么时候掉下来我们都不知道，许云波也不知道，但是他的压力大，他必须要考虑把GamesTV做大后该去往何方，不然就没必要做了。

　　就在这个时候许云波认识了在上海文广工作的中学校友张越，经过张越的牵线搭桥认识了后来的上海文广互动电视有限公司总经理高越（已离职），以及现在的SMG副总裁张大钟。

　　这里要介绍下“上海文广互动电视有限公司SiTV”

　　SiTV是做数字电视节目的，就是需要用电视机顶盒收看的节目，当时旗下有一个游戏风云频道，正好需要大量的游戏节目。

　　一面是GamesTV有人、有钱、有节目却正在发愁没有许可证和播出渠道，一面是SiTV有渠道、有模式、却缺游戏节目、缺专业人才、缺钱，于是，游戏风云GamesTV的结合就成了顺理成章的事情。

　　这里要提一下游戏风云之前的那句口号“游我所爱，任我风云”是有文学硕士学历的高越老师在一次开会时脱口而出的，可见，能在电视台里混出头的人，都不简单。

　　公司合并很快就完成了，游戏风云GamesTV的人员也从十来个人一下子增加到了近50个人，之前的办公室坐不下了，公司搬到了位于广中西路的幻维多媒体谷，这里有大大小小多个专业的演播厅，那个时期比较火的综艺节目，比如，加油好男儿，以及一些明星的见面会等等都是在那里录制的，公司楼下经常被几百个疯狂的粉丝堵的水泄不通。

　　从各方面来说，当时的游戏风云GamesTV真是顺风顺水，一片坦途。

　　下面要介绍一个关键人物了

　　陈剑书，上海文广互动SiTV游戏风云事业部总经理、G联赛创始人

　　2006年6月公司合并后，作为文广方面代表的陈剑书就加入游戏风云GamesTV，他是现游戏风云的总经理，我们喜欢叫他剑书，这也是个很传奇的人。英国硕士毕业，很喜欢游戏，回国后进了文广负责游戏风云频道，刚开始时，频道就他一个人，找片子编片子写策划，硬是撑了很久。

　　陈剑书的背景比较敏感，他自己非常低调，我就不多说了，我只能说他对官场很熟悉，对游戏也十分有兴趣，他的个性其实很强，但是他身上有一种天赋，知道什么时候收什么时候放，知道在电视台这样的国企中如何生存和发展，并且游刃有余，这让他一边能跟文广的上层通上关系，一边知道如何拉拢游戏人才。要知道官场中的分寸是很难把握的，领导不是人干的，有时候面对多方面的矛盾和需求，除了战略方向上正确外，还要经常和稀泥、打擦边球做事，这些可都是大学问。

　　2008年之后的游戏风云正因为有了陈剑书等这样的领导，在中规中矩的电视台里争取到很多开创性的尝试，他和妖魔一个负责对上一个负责对下，在这方面的配合已经炉火纯青，2010年左右游戏风云在赛事和网页游戏方面的成功盈利，离不开他的努力。

　　书归前说

　　2007年1月G联赛正式开始运作了，全国电子竞技电视联赛，这是中国电竞人想了很多年的一个东西，因为韩国数字电视台从1999年星际争霸时代就已经通过电视联赛造星了，并且大获成功，换句话说，这是我们多年来能真实看到，并唯一都认可的最佳的电子竞技模式。

　　G联赛如何好，我就不细说了，大家都能看到，准时、正规、规模大、奖金高并且按时发、场地知名（多次在东方明珠和南京路步行街成功举行），多次与国际电子竞技组织联手举办大型赛事等等。

　　我直接说说G联赛这几年遇到的问题吧

　　1、韩国电视台的盈利模式在近期的中国可能效果不大

　　韩国本身不大，宽带网络和数字电视的普及率又高，这意味着付费率和收视率本身就比较高，收视率高意味着广告商愿意付钱，所以电视台只要做好节目，光是月租和广告费就很可观了。

　　而在中国就不一样了，由于这么多年免费看电视成了大众的习惯，于是，收费数字机顶盒的普及到现在都很缓慢，即使有些大城市已经可以免费收看游戏风云频道了，但是大多数地区依然需要交费后才能开通，这造成了一个很尬尴的局面，让数字电视一直处于红而不紫的阶段。

　　于是，G联赛在前3年的招商中遇到相当多的麻烦，直到2009年都还在烧钱，那段时间G联赛的负责人妖魔的压力相当大。后来随着普及率的改善，以及知名度的打开，赞助开始越来越多，现在这个模式已经形成良性循环了。

　　2、联赛模式有优点也有缺点

　　联赛的选拔是一个相当长的过程，优点是对于赞助商来说品牌曝光度相对较高，益于培养用户的收看习惯，但是缺点也很明显，比赛项目很多，参赛队伍很多，造成看点的分散。

　　3、工作人员疲于奔命，精力消耗

　　我是从游戏风云出来的，我知道电视台的工作压力，在以前每个人每星期是要完成固定时长的节目时间的，而且这些节目并不是录个比赛解说下就结束了，是要有深度，要自己选题、报题、找资料、去拍、去采访、后期编辑、送审，直到审片老师觉得不用改了才算完成的，而且节目是要提前2个星期完成，送到台里还要审。

　　如果遇到节假日之类的，别人都会高兴，我们就会想哭，因为要提前把放假期间老师不能审的片子也做好，不然审片老师来不及审就会开天窗，这会是很严重的播出事故，是要走人的。

　　于是，每个人每个星期连吃饭睡觉，都时时刻刻在脑子里转选题，这个时候如果再多出来一个比赛或者临时的活动，就要消耗更多的精力和时间，所以做时间长了以后，只能有所取舍，集中精力最想做的那期节目，其它的事情就只求稳了。

　　游戏风云和G联赛前几年给人的印象就是稳，规规矩矩，央视风格，就像春晚一样没什么太大的变化和突破，那个时候，我个人认为这是游戏风云需要解决的最大的问题——需要更多的活力和勇气。

　　很让人高兴的是，3月9日在奔驰馆的这场就是一个很大的突破，所以消息一出来我就激动的马上跑去洛川东路的游戏风云公司看看大家的精神面貌，我个人十分期待这场秀，因为我觉得游戏风云的经验积累已经达到一个质变的临界点了，下一站是否是天后，就看这一步了。

　　游戏风云还有很多同事不能一一提到了，比如当时卧薪尝胆的海涛，努力上进的导演王冠，我的好朋友编导Garry，网站部主管孙俊，这些都是最早的一批游戏风云人了。还有很多离开的但是还在圈内的，比如ACE联盟现在的负责人李君，当初也是电竞部的编导。

　　G联赛就像我们的孩子，我们看着她出生、成长，虽然我们忙别的事情离开了，但我们依然欣喜的看到吾家有女初长成，并期望将来看到她的开花结果。
