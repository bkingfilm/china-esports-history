# 十七 EVA剑心补丁包十年造

> 首发于知乎专栏（2013-12-06）原文链接：https://zhuanlan.zhihu.com/p/19631937

中国电子竞技幕后史十七 EVA剑心补丁包十年造

　　玩《魔兽争霸3》（WAR3）对战SOLO的人可能不会玩DOTA，玩DOTA的人也可能不会玩对战SOLO。但是无论你玩哪个模式，这十年只要你玩《魔兽争霸3》，EVA剑心的补丁包都会如影随行。

　　那么，EVA剑心是谁？这位著名的电子竞技幕后工作者是怎么走上这条路的？是一帆风顺还是曲奇波折？现在又在做什么？本期中国电子竞技幕后史，将为您讲述EVA剑心补丁包这十年的故事。

　　EVA剑心（刘继光），1984年1月19日出生于广东韶关，因为家境不是太好，所以从2003年上大一时就经常写些WAR3的战报稿子投给《大众软件》和CBI《电脑商情报》赚点生活费，穷人的孩子早当家，这也铸就了EVA剑心的勤奋和执着。

　　EVA剑心接触WAR3很早，从2002年到2003年他就在KinK（原名翁乐腾，前星际学院版主，War3CN创始人）的[http://War3cn.com](http://link.zhihu.com/?target=http%3A//War3cn.com)上学习，大一时加入了暗黑王子创立的[http://War3xp.com](http://link.zhihu.com/?target=http%3A//War3xp.com)，负责首页的更新。

　　EVA剑心对技术类的事物比较感兴趣，高中时为了给自己的战队做网站，就去自学了Frontpage软件，大一时注意到很多人对WAR3游戏程序如何升级汉化有很大的帮助需求，就开始研究补丁这种程序的制作。

　　虽然EVA剑心是计算机专业的，但是作为一名刚上大一的新生，离反编译和程序开发还是有点距离的，于是他用了一个很笨的方法去了解程序代码功能，这个方法就是先在网上用google查War3每个文件是干嘛的，但是能查到的不仅少而且都是专业性很强的英文资料，对于大多数查不到，他就只能去反复对比程序文件修改前后产生的实际效果，以此来推断每个程序文件的具体功能。即使到今天，WAR3程序的说明文件在网上都不多，所以可想而知EVA剑心在最初阶段花了多少时间和精力。

　　就这样，EVA剑心从2003年的WAR3 1.10版本开始做补丁包，因为更新的比较及时，而且使用方法简便，到了WAR3 1.16版本时，同时期的其它补丁已经很少更新了，大家几乎都在用EVA剑心补丁。

　　到了2004年，暴雪正式推出冰封王座1.18版本后，玩家通过“WAR3地图编辑器”自制了大量RPG地图，由于RPG地图的玩法比SOLO对战的上手难度更低更休闲，同时，EVA剑心的补丁包基于暴雪官方汉化版做了很多补充，这使得游戏门槛被大幅度降低，玩家数量激增，EVA剑心补丁赶上这波热潮，开始大规模普及。

　　这里要插一句关于War3汉化背后的故事，在暴雪推出官方汉化版之前，民间汉化组对WAR3的推广贡献巨大，比如，宽宽汉化小组，这是一个非常有名的游戏汉化组，做过《星际争霸》、《红色警报》、《文明》等游戏的汉化补丁包，对后来的游戏文化影响深远。还有NGA上的著名版主麦德三世（这是个女人），由于她对魔兽世界原著的深刻了解，所以对WAR3剧情战役部分的翻译比较准确详细，受到了各方面的肯定。WAR3以及之后的魔兽世界WOW文化能够沉淀发展，背后有着无数汉化者的不断努力。

　　虽然这里不能一一提到这些幕后工作者，但是我要说中国电子竞技能发展到今天，不仅仅只有选手和解说的汗水，即使相当多的幕后工作者连名字和网站都已经消失了，但是历史不会也不应该忘记这些人的贡献。

　　竞星网的试炼

　　2005年，有两个拿到赞助的人找到EVA剑心，希望做一个有图文内容和游戏视频的全新的电子竞技网站，虽然只有2000块工资，但是模式听起来很不错，而且能做自己感兴趣的事情，这是EVA剑心一直梦寐以求的，他接受了邀请，一切从零开始了。

　　EVA剑心说，2005年是他学到东西最多的一年。

　　新公司开在广州，最开始不到10个人左右，EVA剑心不但要亲自做网页，盯网站内容，还要学习服务器维护和管理，当时的网站服务器在广州地王广场旁边一个电信机房里，他要经常跑去机房处理技术问题，公司还做了视频直播的内容，找来了msjoy和小色，做了《魔兽之夜》、《星际之夜》和《魔兽十大英雄》等节目，为了推视频直播流，还让老板买回来一个推视频流的机器，但是说明书是全英文的，他们又没人接触过这个，只好边硬啃英文边一个按钮一个按钮的去尝试。

　　说起那段时光EVA剑心觉得，虽然公司并不赚钱，还很辛苦，要做很多落地活动才能勉强支撑，但是他依然很开心，因为每天都能学到很多东西，同事们也很有干劲，经常中午一起凑钱到烧腊店买饭吃，边吃边讨论怎么把事情做好。这个时候EVA剑心补丁的市场占有率已经相当大了，所以依靠补丁的用户导入，网站开始有了些关注。

　　可是，现实依然是十分残酷的，虽然模式方向是对的，也有地利人和，但是天时不对这事就很难有个好结果了，即使是从8年后的现在来看，视频点播和视频直播有盈利也是从2010年年底解说与淘宝的深度整合结合之后才起步的事情了，再看2005年左右的大环境，优酷、PPS、PPLIVE这些视频平台才刚刚起步，于是，EVA剑心他们做的竞星网在2006年时就撑不住了。

　　竞星网的失败对EVA剑心打击比较大

　　当时EVA剑心对电竞这行非常迷茫，可以说是心灰意冷，他放下了电竞的所有事情，包括WAR3补丁的更新，回家待了一段时间，2006年7月跟着一个兄弟去珠海做电脑硬件生意了。

　　离开梦想的日子，EVA剑心还没有时间伤感，因为在电脑城里做生意是非常忙碌的，每天早上6点起床忙到晚上12点，虽然买电脑的人很多，现金流看起来很不错，但是因为电脑城里的竞争十分激烈，每台机器的利润非常低，再加上自己也没什么做生意的经验，做了半年发现总帐是亏的，再做下去就没钱进货了，也快过年了，于是EVA剑心就结束掉生意回家了。

　　2007年年初，WAR3 1.21版本推出了，这是一次非常重大的版本更新，游戏的平衡方面有了非常多的变化，而且暴雪更改了CDK的验证方式，也就是说，在这之前的所有补丁技术都全部失效了。无数人涌到EVA剑心的博客和论坛里留言，希望他能尽快推出新的版本补丁，EVA剑心那时还在电脑城里工作，白天太忙没时间看，只能在每天打烊后看看这些留言，面对越来越多召唤的声音，他突然意识到补丁已经变成了自己的一个孩子，这是一种责任，不能放下，于是连夜做好放出去。

　　长辈的支持

　　2007年过年的时候，EVA剑心的意志还很消沉，与长辈们聊天的时候说起过去这些年在电子竞技和电脑硬件生意上失败的事情，觉得十分迷茫，他本以为长辈会劝他放下，但是让他意外的是长辈们说了很多自己的经历让他不要放弃，这给EVA剑心很大的鼓励，开始走出困惑的阴霾，积极去联系新的工作。

　　再次启程，偶游网

　　2007年3月，EVA剑心在武汉偶游网创始人朱学宝的邀请下来到武汉，负责WAR3和星际内容方面的工作，这个时候的EVA剑心已经是在人生谷底，到武汉买完生活用品身上只剩5毛钱了，第一个月吃饭都是朋友请客撑过来的。偶游当时整个公司不到10个人，主要还是依靠一些线下的活动来维持公司，EVA剑心印象最深的是在武汉光谷广场做的WSVG大赛，第一届就加入了WOW竞技场的比赛，现场被挤的水泄不通，效果很好。

　　U9网的邀约

　　也就是在差不多同一时期，EVA剑心因为补丁和人脉方面的资源被U9网的CEO刘亮看中，也对EVA剑心发出了邀请，但是EVA剑心觉得自己刚到偶游网就离开这不好，于是就婉拒了，刘亮始终没有放弃，不但亲自找EVA剑心谈心，还找到了EVA剑心的父亲做工作，这里插一句，从这件小事上就可以看出刘亮的做事方法真是不太一般，这也是他把U9带上一个新高度的原因，他的故事我们会在另一章里详细讲，这里不展开。

　　书接上回，2008年4月，EVA剑心一方面被刘亮持续一年的邀约感动了，另一方面通过几次的接触，他也感受到U9这20多个员工是一帮做事的人，于是EVA剑心来到了西安加入U9公司，负责单机站、WAR3补丁和攻略，以及RPG方面的工作，那时U9的WAR3和RPG方面就有200万的PV，刚刚兴起的DOTA站已经有20多万PV了。

　　EVA剑心与U9结缘已久

　　早在2004年U9还是一个个人站的时候，创始人灰磷问EVA剑心如果专门做一个RPG的论坛觉得如何？EVA剑心说，虽然现在RPG还不是主流电竞项目，但是未来应该有前景。

　　2008年，EVA剑心开始着手U9 war3方面以及竞技类网站的组建，2009年EVA剑心成为单机组主管，2010年做了高级经理，2011年内容部副总监，经过三年的艰苦经营，U9的war3类站点已经成为业内公认的第一大站，2010年U9的DOTA站做到了1000万PV的高峰。这离不开他带着一支精英的团队。

　　回顾EVA剑心2008年到2011年在U9的工作，可以总结为四点：

　　一、通过补丁和攻略站把War3 RPG带到一个新的高度，当时全站有70%左右的流量来自单机组。

　　二、通过DOTA模拟器和资讯内容把DOTA站带到了1000万PV。

　　每次DOTA版本更新，U9总能快速更进攻略和模拟器，甚至可以从冰蛙哪里优先获得资料，让玩家在最短的时间获得最新版本的信息。

　　三、在2009年就带团队切入美服的英雄联盟，成为国内最早的英雄联盟玩家聚集地，并推出U9超级助手插件，做了美服汉化，方便玩家的使用。

　　2010年TGA，美国英雄联盟项目组来到上海，EVA剑心亲自采访了羊刀等几个国外的项目负责人，跟他们深入的聊了3个多小时，发现了很多有趣的事情。

　　大家知道英雄联盟在DOTA最盛行的时候，十分大胆的没有照抄DOTA，相反他们做了很多的改变，比如去掉了反补、设计了野区BUFF、可以隐身的草丛、死了不掉钱等等，他们的目的是什么？是无意的尝试还是有意的改变？

　　在这次采访中，EVA剑心第一次获得了真实的答案，他发觉LOL整个项目组都对游戏非常的熟悉，他们做的这些设计就是很明确的为了缩短高手和新手之间的差距，不让玩家有很大的挫败感。

　　EVA剑心的这次采访报道让国内玩家第一次明确的了解英雄联盟的设计理念和未来的发展方向。LOL项目组也十分感谢这次专业的采访拉近了和中国玩家之间的距离。

　　四、把U9星际争霸2站做到了100万PV。

　　EVA剑心说，在U9的这几年最大的收获是学会如何去带好一个团队。他的升职也很快，到2011年，他升任U9副总监，管理十几个单机组的工作。

　　可是，家家有本难念的经，U9的这本也很复杂，虽然从2009年正式转型综合性游戏网站，也成功的推出第一款网游DNF站，之后的网站收入一直是个问题，再加上公司在西安，顶尖人才招不到也留不住，EVA剑心也很无奈，处于最好的时代却使不上劲的感觉是很令人沮丧的。

　　2011年，EVA剑心觉得自己年龄大了，家人也希望他能到离家比较近的地方工作，于是，他接受了广州多玩游戏的邀请，回到了广州负责新网游的内容，重新开始。

　　EVA剑心说他要感谢的人很多，特别是U9的刘亮和吴峰，在事业和生活方面给予了很多指导和帮助。一起走过来的团队成员和兄弟们是他最大的财富。

　　EVA剑心刚有了孩子，他给我看照片时十分开心，我看到这一幕，突然意识到写电子竞技幕后史前后采访的这近50位电竞人，基本上都是刚有孩子不久，成家立业，终于轮到我们这批80后电竞人了，还有电竞人打趣的说，我们将来可以弄个“电竞人亲子聚会”，反正年龄都差不多，大人们叙叙旧，孩子们也能成朋友。

　　我相信有一天会实现这个想法的。
