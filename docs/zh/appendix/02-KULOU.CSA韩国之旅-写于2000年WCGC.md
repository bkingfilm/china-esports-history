# KULOU.CSA韩国之旅 写于2000年WCGC

> 首发于知乎专栏（2016-11-18）原文链接：https://zhuanlan.zhihu.com/p/23774762

文/kulou.csa 写于2000年参加完WCGC之后

　　初秋，中国星际代表队的四位选手将代表中国出征在韩国举行的WCGC大赛。

　　6日，在首都机场，我们登上了前往汉城的飞机，经过一个半小时的飞行，我们抵达了韩国。

　　大赛组委会派了专人在机场迎接中国代表队，组委会非常细心，接待我们的是一位懂中文的小姐，一阵寒暄，我们上了外面印有World Cyber Game Challange的大吧。

　　在车上，接待员告诉我们，晚上组织者会在汉城著名的shila饭店举办一个盛大的酒会欢迎所有来参赛的选手，并且很热心的教起我们韩文来，虽然我们是学了这句忘了那句，但还是很感激接待员的热情。

　　到了饭店，因为组织者安排其他选手到汉城的一个公园游玩去了，所以暂时只有一些晚到的选手在场，而我们正闲逛的时候，见几个工作人员扛着几个大牌子走出饭店，仔细一看，正是大赛抽签以后的分组情况，于是我们冲出饭店，追着大牌子跑，总算工作人员在一辆大卡车前停了下来。

　　于是各自看了看自己的分组情况，有喜有忧，不过毕竟都是经过重重预选赛才到韩国的，所以无论碰上谁应该都不好对付。

　　不久，所有选手都回到了饭店，只见整个饭店立刻变成了红色的海洋，因为选手们都穿着组委会分发的印有World Cyber Games的红色比赛服，当时的场面盛是壮观。

　　皆是驰名于战网上的绝顶高手，此刻离我们这么接近，心情不由激动起来。能与他们同场较量，真是很幸运，心里也不由暗想，一定要珍惜这个难得的机会，让国外著名的高手也知道中国。

　　晚上，酒会正式开始，其间还安排了几个节目如唱歌跳舞什么的，并有EA亚洲的总裁在会场发言，最后，在鼓声中酒会正式结束了。

　　酒会结束后，大巴车把选手带到住宿的地方。

　　住的地方是一个度假村，环境相当幽静，条件也很不错，我们星际代表队住在三层最角落，房间内有些家具，还有电视，电话，卫生间，而且还有橱具，可以自己煮东西吃，而火炕，则让你绝对不会着凉。：）

　　内部还有两间电脑屋，供选手训练用。可惜电脑少了点，不能人手一台，不过在旁边观看国外高手之间的对战也挺不错。

　　各国选手很快就互相打成一片，虽然语言方面有些障碍，但我们都是狂热的游戏玩家，语言阻止不了我们之间的交流，感觉国外高手很容易沟通，并不会摆架子什么的，我们也就乘这个机会和国外高手切磋几局。

　　不过鏖战正欢的时候，组织者让我们休息了，也是，刚下飞机就参加了酒会，是有点累了，而且比赛还有很多天，有的是机会和高手们切磋，于是我们就回房睡觉了。心中期待着明天快点到来。

　　一觉醒来，已经是日上三竿，房间里的广播催着选手们到楼下集合，马上要去比赛场地了，于是匆匆准备就绪乘上了开往比赛场地的大吧。从住地到比赛场地大概45分钟车程，很快我们便来到了赛场。

　　比赛是在韩国最大的游乐场Everland举行，里面娱乐设施齐全，并且我们来比赛的选手可以免费游玩，这样选手们可以在紧张的比赛过后，在游乐园内好好放松放松。

　　今天只是开幕式，没有正式比赛。

　　我们先是被带到一个Everland里的网吧，网吧非常的豪华，和国内的网吧差别无法言表，里面有将近200台机器，并且分上下两城，在中央位置放着四台机器，然后其他所有机器是环绕着这四台机器摆放的。

　　网吧里经常有韩国的职业玩家在里面进行训练比赛，中央的四台机器就是他们训练专用的，而且旁边的墙上还挂着一个超大的屏幕，负责直播职业选手之间的训练，别人则可以通过大屏幕观看学习，这些都是在后面几天比赛时看到的。

　　在上了会儿网后，到了午餐时间，午餐在游乐圆内的意大利餐馆进行，而且比赛期间，午餐和晚餐都在游乐园的餐馆吃，也许是因为来参赛的选手大多为欧美国家的选手，大赛为选手提供的均为西餐，这样我们中国玩家就非常的不习惯，多数中国选手都说吃的不怎么好。

　　午餐结束后，我们来到了正式比赛的场地，Everland里的剧院，内有几百个观众席位，舞台左右有两个大屏幕为观众直播精彩的比赛。

　　下午这里将要举行开幕式，所以舞台中央并没有摆上机器，经过等待，WCGC开幕式正式开始，让选手们欣赏了很多精彩的歌舞，这在国内游戏比赛中肯定不会有的，韩国方面还邀请了韩国最著名的男歌星登台为选手演唱，并且甚至请到了韩国文化部的部长在开幕式上致辞。

　　文化部长亲临游戏比赛现场，可见游戏在韩国的地位非比寻常。

　　开幕式完了之后，我们又参观了组委会开设的几个展区，参观之后今天的活动就结束了，大巴又将所有选手带回了度假村，一下车，只见选手们如猛虎一般奔向了电脑屋，生怕迟了抢不到机器，而我们玩星际的几个慢悠悠的走到电脑屋时，里面已经是挤的水泻不通了。

　　空机子不用说自然是没有啦，我们只得在一旁观看高手的较量，韩国的几位选手比较受关注，他们后面站满了选手围观，我的感觉是韩国的几位手快的吓人，几乎可以用快如闪电形容，而其他国家选手一般都比较慢，不知何故：）

　　由于第二天还有比赛，我们看了几场就休息去了，有点紧张却也有点期待，毕竟是第一次有机会参加这样大的国际比赛。

　　10月8日，WCGC终于拉开了正式比赛的序幕。

　　比赛是这样安排的，星际争霸和QUAKE3一天，FIFA2000和AOE2一天，四个游戏隔天举行，直到比赛结束，这样比赛时间非常的充足，每位选手每天的比赛一般不会超过三场，选手能够有时间进行休息，状态的调整等，能保证比赛的精彩程度。毕竟比赛是否精彩是很重要的。

　　第一天是星际争霸和雷神之锥，所以帝国和FIFA的八位兄弟并未和我们一起出发，他们将留在度假村进行训练，而我们星际和雷神的八位战士都整装待发。

　　11点，我们和领队acep上了大吧，与我们坐同一辆大吧的正是韩国队的选手，他们在车上都谈笑风生，而我们中国的几位都很严肃的若有所思状，大概因为都是第一次参加这样的比赛，有些紧张。

　　到达Everland里的比赛场后，先是就餐，不多说了:)，下午比赛即将开始，在剧院的舞台上摆了30多台机器，先是20分钟时间给选手们试机器，这才有机会知道比赛用机配置怎样。

　　比赛的机器相当的好，均为P3-800，256兆内存，玩QUAKE3时能够达到130帧左右，当然其他游戏选手对机器并不怎样要求，主要就是键盘鼠标好使就行了，而且大部分选手都是自带鼠标的，所以对比赛的机器，选手基本没有什么意见，我没有自己的鼠标，使用赛场提供的，感觉都也很好用。

　　比赛组织者比较友好，对每位选手都是有问必答，而且专门请了懂英语和中文的工作人员在比赛现场维持比赛，免了因为语言不通造成比赛不能很好的进行。试机时有说着比较流利的汉语的小姐来问你是否有问题。

　　试机之后就是正式比赛了，比赛中HeNRY_HO的机器由于USB驱动问题，有些键不怎么灵光，结果被台湾和法国选手各赢一盘，遗憾的被淘汰。

　　而PBY.csa ID_eat各赢一场输一场晋级32强，我赢了两场也晋级了32强。

　　雷神方面除了ID_CHJ因为小组对手实力都很强，没能通过第一轮，而其他三位选手均顺利晋级32强。

　　战绩算是很不错了。

　　晚上回到度假村，因为9日没有星际的比赛，所以我们玩到电脑屋关门才走，不过12点就关门了，早了点:)。

　　9号一天都待在度假村里训练，晚上传来消息，帝国和FIFA的选手全部通过第一轮，虽然不是玩同一个游戏，但也替他们高兴。

　　但在10-13号这几天的比赛中，我们的选手相继落马，到13号，所有选手均被淘汰了，没有人能进前四强，拿到奖金。

　　这毕竟是我们第一次出征国际大赛，能有现在这样的成绩，已经是不错了。非常遗憾的是我在星际争霸中只停留在12强，没能取得更好的成绩，我向大家表示歉意。

　　因为比赛会有选手不断被淘汰，所以大赛组织了到汉城旅游的活动，10-14号每天都有三个地方供选手选择，有风俗区，电子市场等。

　　不过由于直到12号我都有比赛，没去成，而被淘汰后，又没有心情出去玩了。所以我一天也没去，其他几位选手都去了，个个都采购了一大包东西。

　　11号晚上，组织者还办了一个Party，并有自助餐，而且有韩国烧烤，早听说韩国烧烤味道极佳，这次总算见到正宗，感觉味道的确很好，就餐后所有选手在度假村的广场狂欢，很多国家的选手登台跳起了颈舞，甚是搞笑，愉快的一晚。

　　14日我们因为都已经被淘汰，所以星际的所有选手和acep都到Everland旁边的游泳池游玩了一圈。放松心情后，对失败也看开了一些，调整心态，准备迎接以后的挑战。

　　14号和15号是决赛的日子，也是韩国选手的日子，在两天四个游戏的决赛中，韩国选手欢呼了三次胜利，他们除了在雷神中全面失利以外，包揽了其他三个游戏的冠军。

　　并且其中尤以FIFA最为夸张，韩国人包揽了前四名。星际争霸中前四占了三个，帝国也有两名韩国人在前四。职业和业余的差距摆在了我们的眼前。

　　15号晚，所有项目均已结束。接着的是颁奖仪式，几位头面人物为各个游戏的获胜者带上了金牌，银牌和铜牌，并颁发了大支票和鲜花。而且每个游戏发完奖以后，大屏幕都会出现冠军所在国家的国旗，并会奏起国歌。

　　感觉就和刚结束的奥运会颁奖差不多，难怪外界对WORLDCYBER GAME CHALLENGE有游戏奥运会之称。

　　看着领奖台上的兴高采烈的选手，难免有点心酸，如果能有中国人站在领奖台上，那该有多好啊。

　　但只能期待下次WCGC，中国选手能给大家带来惊喜了。颁奖仪式很快结束了，组织者在Everland的一处广场搞了一个欢送晚会，又是一顿自助餐，也有韩国烧烤，都是韩国师傅现场烧烤的，虽然当天晚上温度很低，但选手们还是吃的痛快。

　　酒足饭饱后，又有韩国的舞蹈欣赏。而到最后大家更都全部涌到舞台上，跳起了disco。

　　这时候，选手之间已经没有比赛，没有竞争，有的只是友谊和快乐，大家乘着最后的时刻互相留下联系方法并且拍照留念等，我也找了I_love.star和GoRuSh合了影，和冠军亚军留个纪念。

　　回度假村前，Everland的上空放起了光彩夺目的礼花。而回到度假村后，还有很多选手聚在房间内，通宵畅饮，道不尽依依惜别之情。

　　10天的韩国之行，让我充分感受到韩国人对游戏的热情，以及韩国人对游戏的理解。

　　像本次比赛在Everland的中央地带有设大屏幕显示器，只要比赛开始，就不断播放比赛的广告什么的，每位来游乐场玩的游客一般都会看到。而且还专门搭了一个舞台，礼仪小组成天在不断向游客介绍WCGC，分发传单等。

　　比赛在Everland的剧场里举行时，下面经常聚满了对游戏感兴趣的游客观看比赛。最后的决赛更是令观众席座无虚席。而且我们在韩国时经常能在电视上看到现场直播星际争霸的比赛，并且一次我看到一对三十几岁的夫妇进行2v2的比赛。

　　在韩国玩家的年龄跨度非常大，四十五十玩游戏一点也不稀奇，而在中国很少能看到超过三十的玩家。而且韩国的女性玩家非常多，电视上看的星际比赛几乎都是女子的，但在中国的女子玩家恐怕都被像珍奇动物一样看待。

　　唉，中国玩家何时能盼到春天？

　　WCGC大赛结束了，但我们永远记的韩国，永远记的在韩国的日子。
